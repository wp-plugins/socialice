Socialice is a widget that will display social media icons in your sidebar.

== Description ==

Socialice is a sidebarwidget that displays several social media icons. Visitors can click on the icons and they go to your Facebook, Twitter or Google+.

== Installation ==

1. Upload `socialicebuttons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change settings in settings > Socialice.

== Frequently asked questions ==

= What social media can I use this for? =

Currently the widget supports links to your Facebook, Twitter and Google+
This will be expanded soon.
